## ローカルでの実行方法

node 14 以降が必要です。また `yarn` の利用を推奨しています

インストール・セットアップ

```
npm i -g yarn
yarn install
```

サーバの起動。下記コマンド実行後、localhost:8020 にアクセスしてみてください。

```
yarn dev
```
